# voice-ui

Voice UI integration scripts — text-to-speech and speech-to-text pipelines for OpenClaw. Supports Fish Audio TTS with voice cloning reference IDs.

## Topics
python, voice, tts, stt, audio, openclaw

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/voice-ui.git
cd voice-ui

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 voice-ui.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
